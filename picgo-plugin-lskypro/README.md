# picgo-plugin-lskypro

plugin for [PicGo](https://github.com/Molunerfinn/PicGo)

- LskyPro Uploader

## 使用

### 配置

- Url：使用LskyPro的图床的API上传网址
- Token：在图床获取到的Token
- 如果出现问题请联系Weifeng
- 博客：[Weifeng's Blog](https://wfblog.net/)
